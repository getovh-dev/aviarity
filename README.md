# ✨ Aviarity — автоматизация BIRD2

> Генерация и безопасное применение конфигов BIRD2 через YAML + Jinja2

---

## 🚀 Описание

**Aviarity** — это инструмент для автоматизации BGP-конфигураций в BIRD2.

Вместо ручного редактирования огромных конфигов ты:

- описываешь сеть в YAML
- генерируешь конфиг через шаблон
- проверяешь перед применением
- автоматически откатываешься при ошибке

---

## 🧠 Возможности

- YAML как источник правды
- Jinja2 шаблоны
- Diff перед применением
- Валидация через `bird -p`
- Автоматические бэкапы
- Откат при ошибке
- Быстрое применение конфигурации

## 🛠 Установка

```
git clone https://github.com/getovh-dev/aviarity
cd aviarity

mkdir -p /opt/aviarity/templates

cp generate.py /opt/aviarity/
cp peers.yaml /opt/aviarity/
cp templates/bird.conf.j2 /opt/aviarity/templates/
```

## 🧾 Пример конфигурации (peers.yaml)

> ⚠️ ОБЯЗАТЕЛЬНО замени все IP, ASN и префиксы на свои

```
router_id: "1.1.1.1"
own_as: 65001
kernel_prefsrc: "2001:db8::1"

own_prefixes:
  - "2001:db8:100::/48"

upstreams:
  - name: "upstream1"
    description: "Example Transit"
    remote_as: 64500
    local_ip: "2001:db8:1::2"
    neighbor_ip: "2001:db8:1::1"
    export_prefixes:
      - "2001:db8:100::/48"

downstreams: []
byoip: []
ixps: []
collectors: []
static_announces: []
```

## ⚡ Использование

### Сгенерировать конфиг (без применения)

```
python3 generate.py --dry-run
```

### Посмотреть изменения (diff)

```
python3 generate.py --diff
```

### Применить конфиг

```
python3 generate.py
```

---

## 🔒 Безопасность

Aviarity делает применение безопасным:

1. Создаёт бэкап текущего конфига  
2. Генерирует новый конфиг  
3. Проверяет через `bird -p`  
4. Применяет через `birdc configure`  
5. При ошибке автоматически откатывается  

---

## 🔁 Бэкапы

Хранятся в:

```
/etc/bird/backups/
```

- Хранится 10 последних конфигов
- Старые удаляются автоматически

## 📢 Обновления

⚠️ Не вносите изменения напрямую в:

- `bird.conf`
- `templates/bird.conf.j2`

Эти файлы управляются Aviarity и могут быть перезаписаны при следующей генерации.
